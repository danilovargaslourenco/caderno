Aprendendo Sobre Pseudo-Classes

Div

Para entendermos as pseudo-classes, precisamos entender brevemente sobre as div, elas sao apenas um area, uma linha que segue ate o final da pagina. Podemos mudar a altura e largura das div's com as Propriedades de Estilo "Height" e "Width", também conseguimos mudar a posição dos textos nas div's com a Propriedade text-align, sendo que podemos adicionar bordas às div's com a Propriedade "border" com o valor "solid".Por padrão as div's ficam alinhadas uma em cima da outra, mas se quisermos que fiquem uma do lado da outra, devemos usar a Propriedade de Estilo "display" com o valor "inline-block", que seria um alinhamento de bloco em linha, assim fica tudo na mesma linha. Segue o exemplo abaixo

div 01
div 02
div 03

Segue o codigo: 
<div>div 01</div>
<div>div 02</div>
<div>div 03</div>

E, sem a tag "inline-block" elas ficam assim:
div 01
div 02
div 03
Voce pode dar id as divs ou classes tambem.

Pseudo-Classes

As pseudo-classses se referem a um estado especial de um elemento, se voce quiser dar uma acao especial a ele, voce pode usar diferentes maneiras para determinar essa mudanca. Nos referimos a uma pseudo-class com ":", como exemplo iremos usar a tag hover e visited, segue abaixo:

Nesse exemlo abaixo, criaremos uma div, com background-color e com border, e quando passaremos o mouse em cima, ela mudara de cor:

Hello, World!

CSS  -->  change:hover { background-color: rgb(0, 70, 70); border: 1px solid white}

HTML -->   <div id="change">Hello, World!</div>

Agora o exemplo do visited, que se refere a algum link que ja foi visitado. No caso quero que ele mude de cor apos ser visitado, nao fique roxo como padrao.

Google.com 

CSS --> a:visited { color: #f013d7;}
a:hover { background-image: linear-gradient(to top, #ff2247, green);}

HTML --> <a href="https://google.com">Google.com</a>

Temos muitas outras variacoes de pseudo-classes tambem, como: :visited, :active, :checked, :empty e :focus.

Sinalizando para tags dentro das div's

Para voce indicar alguma pseudo-classe para alguma tag dentro de uma div, usa se ">", como exemplo: "div > p" voce estaria indicando um paragrafo dentro da div. Um paragrafo filho de uma div.

Pseudo-Elementos

Os pseudo-elementos sao referencias ao conteudo periferico do elemento, como o conteudo antes '::before' ou o conteudo depois '::after', voce pode adicionar qualquer coisa antes ou depois do elemento apenas usando os "::" after ou before.
