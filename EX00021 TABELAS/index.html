Muito importante criarmos tabelas para apresentarmos informacoes e dados para os nossos consumidores. Importante sabermos a hierarquia de tabelas (simples) 

TABLE <tr> = tabela
   TABLE ROW <tr> = linha de tabela 
     TABLE HEADER <th> = cabeçalho de tabela
     TABLE DATA <td> = dados da tabela.
Uma tag de CSS muito importante é o border-collapse: collapse, para as bordas se juntarem e ficar igual uma tabela normal. A configuracao normal e border-collapse: separate. Para as bordas ficarem juntas como o exemplo acima, voce precisara dessa definicao.
Muito importante seguir de maneira correta a hierarquia na hora de escrever seu codigo, porque se voce errar a estrutura, ira baguncar a tabela e nao ficara do jeito que voce quer. Tabelas sao rigidas e nao permitem erros de digitacao.

ALINHAMENTO

Podemos alinhar o conteudo da nossa tabela de 2 maneiras: text-align= para alinhar horizontalmente nosso conteudo, vode pode usar left, center e right. Tambem podemos usar o vertical-align = para alinhar verticalmente, podemos usar o top, middle e bottom.

EFEITO ZEBRADO

Para facilitar a leitura em tabelas grandes e com bastante dados, podemos criar um efeito zebrado, que no caso seria uma linha de cada cor, para isso precisamos usar a pseudo-classe :nth-child(2n), essa pseudo-classe significa que a cada 2 (por isso o 2n, mas pode ser 3n, 4n etc) tags selecionadas elas terao a configuracao que vc escolher. Por exemplo, se voce colocar tr:nth-child(2n) { background-color:black } a cada 2 tr, uma tera background-color black. Podemos usar os termos odd and even, para, respectivamente, impar e par.
