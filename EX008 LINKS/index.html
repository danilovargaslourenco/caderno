Aprendendo sobre Links e Ancoras

Links Externos

Para criar links no seu site, pode usar a tag <a>. Toda tag de link vem com o parametro href que significa a URL na qual voce esta indicando. Segue o exemplo: 

 <a href="url">link text</a>
Existem parametros nos quais voce pode inserir dentro da tag que sao:
  • <target="_blank"> - Para nao abrir outra aba quando o visitante clicar no link
  • <target="_self"> - Para ser aberto na mesma janela.
Tambem existe a um parametro que indica para onde esse link vai. Segue o exemplo:
  • <rel="next"> - Indica que o link mostra a proxima pagina do documento atual.
  • <rel="external"> - Indica que o link sera para outra pagina.
  • <rel="prev"> - Indica que o link mostra a pagina anterior do documento atual.
  • <rel="author"> - Indica o site do autor do artigo atual.
Segue um exemplo de um codigo completo para um link externo aberto em outra aba. 
<a href=URL target"_blank" rel="external">Nome do site que voce quiser colocar</a>

Links Internos

Para indicar pasa uma segunda pagina ou outro website de sua posse, apenas selecione o arquivo dentro do parametro href=. A proxima pagina precisa estar na mesma pasta, assim nao e necessario indicar a URL, apenas selecionar. Links para Download

Para colocar links para download voce precisa usar o parametro download=Seu Arquivo e tambem o type=(codigo especifico para cada tipo de arquivo). Para descobrir o codigo, acesse o site iana.org e busque pelo formato correto do seu arquivo. Segue o exemplo:
<a href="../EX008 LINKS/downloads/Curriculum.pdf" download=Curriculum.pdf type=application/pdf>Baixar Curriculum em PDF>/a> Baixar Curriculum em PDF
