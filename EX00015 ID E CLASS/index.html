Aprendendo sobre os parametros id e class

Quando usar id?

O parametro id vem logo apos qualquer tag que voce queira modificar diferentemente das outras, como por exemplo, voce queira colocar o h1 principal diferente dos outros h1, ou um paragrafo alinhado diferentemente dos demais paragrafos, voce identifica essa tag que voce quer diferenciar, com id=NomeEscolhido, e no CSS voce pode simplesmente se referir a tag usando #NomeEscolhido. Voce pode usar a mesma id apenas para uma tag, esse mesmo id nao pode ser usado varias vezes, portanto, esse parametro e um pouco limitado.Vejamos um exemplo. 

          html  <h1 id=textoprincipal>Ola mundo</h1>
           
          css        h1#textoprincipal {
                       text-align:center}

Quando usar class?

A class pode se tornar muito mais util que a id, pelo fatomdemvoce poder usa-la em diferentes tags para se referir a uma especificacao no CSS. A class e usada da mesma maneira da id, mas para se referir a essa class no CSS voce usa o "." ao inves da "#" da id. Voce tambem pode usar varias class na mesma tag, (separadas apenas por espaco). Veja o exemplo abaixo. 

<h1 class=principais>Titulo 1</h1>
<h1 class=principais>Titulo 2</h1>

    .principais {
        text-align:center
        color: blue}
Nesse caso acima, os 2 h1 selecionados herdarao as configuracoes inseridas dentro dessa class. Na hierarquia do CSS, o id vem primeiro, logo apos o class e por ultimo a definicao geral. Nota: No HTML temos a tag "<span><span>", ela não tem uma função específica, serve apenas para demarcar um trecho de texto caso quisermos fazer algum tipo de alteração de estilo nesse trecho com uma class e etc.
