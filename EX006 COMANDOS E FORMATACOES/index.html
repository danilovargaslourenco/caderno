Comandos e Formatacoes

Codigo fonte/ Pre formatacao

Voce pode usar a tag < code > para se referir a codigos. Com essa tag o codigo descrito ficara monoespacado e mais arrumado visualmente. Exemplo:
mylist = [False, True, False] x = any(mylist)

Mas esse codigo ainda esta na mesma linha e junto, ele nao parece um codigo. Para resolver esse problema usamos a tag <pre> envelopando as tags <code>. Segue o exemplo: 


mylist = [False, True, False]
x = any(mylist)

Citacoes

Curtas Citacoes

Voce pode usar a tag <q> para curtas citacoes como frases e breves falas. Esas tag adiciona aspas no texto. E importante ser usada pois da semantica ao seu site e nao apenas forma. Segue o exemplo.
So sei que nada sei

Longas Citacoes

Para longas citacoes, como paragrafos, ou textos, voce pode usar a tag <blockquote>; para formatar o texto espacadamente e tambem consegue adicionar da onde voce tirou esse texto com o parametro "cite = url" dentro da tag blockquote. Segue o exemplo:

Lorem ipsum dolor sit amet consectetur, adipisicing elit. Magni ex consectetur ipsa commodi autem? Repellat molestias id in aperiam vero recusandae veniam, laboriosam dolorem corporis nemo mollitia, voluptatibus assumenda. Eius id ducimus minima non deleniti!

Abreviacoes

No caso de abreviar, e colocar um pop-up na palavra quando parar o mouse em cima da mesma, colocar um significado em siglas ou em palavras complexas. Para isso voce usa a tag <abbr>. Segue o exemplo abaixo:
Eu estou estudando HTML e CSS.
