Listas

Listas sao muito importantes para um site. Para criar uma ordered list usamos a tag <ol>, e para uma unordered list usamos a tag <ul> para delimitar as listas. Usamos a tag <li> para indicar cada item da lista. A tag <ol> e <ul> podem ser editadas em muitas variedades e formas no CSS, usando os codigos abaixo: 


            <style>
                ul {list-style-type: disc;}
                ol {list-style-type: upper-roman;}
            </style>
