Aprendendo Sobre Imagens Dinamicas

Imagens dinamicas sao imagens que se adaptam de acordo com o tamanho da tela do usuario. Para cada tamanho de tela, podem ser criadas imagens de diferentes tamanhos para melhor se adaptar. Para fazer isso usamos a tag <picture>, e seguindo a ordem da menor para a maior, selecionamos a maior imagem primeiro como imagem padrao, (o ultimo item da tag picture deve ser a imagem padrao, a maior) com a tag <img src="Seu Arquivo" alt="Descricao Semantica">. Acima desse codigo colocaremos a imagem media, ou a segunda maior (Seguindo a ordem da maior para a menor), com a tag : <source media="(max-width: Colocar o tamanho maximo da imagem)" srcset="Selecionar arquivo alternativo para caso a tela for menor que o tamanho anterior" type=image/(Colocar qual tipo de arquivo)>. E assim sucetivamente, caso voce queira usar imagens cada vez menores no site, so seguir o padrao acima, sempre respeitando a ordem e colocando a imagem menor primeiro. Segue o exemplo: 


<picture>
<source media="(max-width:750px )" srcset="imagens/small.png" type="image/png">
<source media="(max-width:1050px)" srcset="imagens/medium.png" type="image/png">
<img src="imagens/Untitled.png" alt="Imagem Grande">
</picture>

Segue o codigo acima em funcionamento. Teste aumentar e diminuir a tela do computador.
