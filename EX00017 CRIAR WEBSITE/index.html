Coisas Importantes para Criacao de um Website

Planejamento

Antes de mais nada precisamos planejar o site, como layout, cores e fontes. Temos a ferramenta chamada MockFlow, para layout, e para cores e fontes ja vimos anteriormente.

Semantica do site

Muito importante se atentar a semantica do site, como cada parte do site que voce criou com o mockflow esta de acordo com as GROUPING TAGS e cada parte organizada corretamente. Interessante colocar lado a lado o esboco de layout e ir seguindo de maneira organizada o seu codigo para ficar facil de ser modificado futuramente. Also important: separar tudo por pasta, dentro da pasta do seu projeto, como pasta de imagens, pasta de fontes e cada item na sua pasta. MANTER O CODIGO ORGANIZADO, tambem facilita bastante a compreensao e o desenvolvimento do site, e tambem para ajustar futuramente.

Variaveis em CSS

Voce pode definir uma configuracao padrao para todas as tags seguintes, como por exemplo: conseguimos definir a cor da nossa paleta como "--cor01", e com isso conseguimos apenas colocar esse nome anterior quando selecionar a cor e para mudar e so mudar na variavel, que todas as tags em que esse nome foi inserido mudara, serve como um nome para uma tag, isso facilita na hora de mudar o codigo futuramente, segue o exemplo de como fazer:

:root {
            --cor01: white;
            --cor02: red;
        }

Agora em todos os seletores de cor que voce selecionar "var(--cor01) tera a cor white, e para alterar, basta altera-la uma vez.

Pode usar tambem para fontes

:root {
            --fontepadrao: Arial, sans-serif;
            --fontecursiva: 'Bambo' (exemplo), cursive;
}

Voce pode selecionar fontes padroes para serem seguidas, e novamente, para ser mudado ou editado, basta ser mudado uma vez, assim facilitando a customizacao do seu site e a edicao do mesmo.

Configuracoes Globais CSS

Caso voce queira selecionar TODOS as tags do seu CSS com uma definicao, por exemplo, margin:0px, ou any definicoes, e simples: 


* {
    margin:0px;
    padding:0px;
}

Nessa configuracao acima, TODAS as tags terao paddin e margin de 0px, isso pode facilitar na hora de customizacao e criacao do site.

Responsividade do site

Para montarmos um site responsivo, precisamos primeiro definir o min-width (o texto so ira se adaptar ate a largura minima de maneira responsiva) e o max-width (a largura maxima possivel do seu site, mesmo que numa tela maior). Essas sao as 2 tags mais comuns para tornar textos e titulos responsivos no seu site.

Responsividade em imagens

Para tornar imagens responsivas no seu site, voce pode usar a tag width: 100%; de maneira mais automatica, mas para ficar mais certinho, voce pode tambem criar imagens menores e diferentes para cada tipo de tamanho do site, como por exemplo se alguem acessar no computador vera uma imagem de tamanho diferente de alguem que esta vendo pelo celular. Para adicionar imagens diferentes de acordo com o tamanho, segue o exemplo: 


>picture<
source media='max-width=800px' srcset='imagem que aparecera quando atingir 800px'
img src='imagem principal ate os 800px' alt='x'
>/picture<
Basta selecionar o max-width e sera quando a segunda imagem ira aparecer, ideal criar outros tamanhos de imagens para a responsividade ficar melhor encaixada no site.
